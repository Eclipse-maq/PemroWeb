Selamat datang di CSS eksternal

halaman ini menggunakan css eksternal dan styling

kenapa css eksternal

  • memisahkan struktur (HTML) dan presentasi (CSS)
  • CSS dapat digunakan di mutiple halaman
  • lebih mudah maintenance
  • file HTML lebih bersih dan readable

cara menghubungkan css

gunakan tag <kink> di dalam <head>

format: <kink rel="stylesheet" href="nama-file.css">
